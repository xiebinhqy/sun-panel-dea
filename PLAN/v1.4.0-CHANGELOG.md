# v1.4.0 版本更新日志

> 发布日期: 2026-04-26
> 版本号: v1.4.0
> 基于 Sun-Panel v1.3.0 二次开发

---

## 🆕 新增功能

### 1. OpenAPI 开放接口（beta）
开发者可以通过 API 接口实现功能扩展（需 Token 鉴权）
- 后续提供详细 API 文档

### 2. 全局站点设置
- 自定义站点标题 PRO
- 自定义站点图标 PRO
- 自定义登录页面背景图

### 3. 在线编辑全局文件
- 在线编辑 index.js 和 index.css（PRO 用户）
- 非 PRO 用户仍可从安装目录修改

### 4. 背景图格式扩展
- 支持 heic、avif 格式上传

---

## 🛠️ 修复

- 修复 v1.4.0-beta 旧版升级不兼容在线编辑 JS/CSS 的问题

---

## ⚡ 优化

- 禁用 referrer，解决 qBittorrent 等跳转问题
- 优化获取三方网站图标
- 编辑项目时网址检测（未以 http/https 开头提醒）
- 修改系统状态详情图标显示格式（硬盘/内存: 已使用量/总量）

---

## 📦 其他

- Docker 版本精简挂载目录为单个 `./conf`
- 仍然兼容旧版本的多路径挂载方式
